# **App Name**: SourceWise AI

## Core Features:

- Intelligent Question Analysis: Analyze user questions to determine the relevant data sources (websites, databases) for providing accurate and contextual answers. This includes natural language understanding and intent recognition.
- Dynamic Data Retrieval: Dynamically retrieve information from preloaded websites (e.g., lvb.cm, matrixtelecoms.com, adac.cm, groupeiccnet.cm) and connected databases based on the analyzed question. Supports scraping websites and querying databases in real-time. Use a tool to decide whether to include snippets of code in its output.
- Natural Language Response Generation: Generate clear, concise, and natural-sounding responses by synthesizing the information retrieved from various data sources.
- Database Management: Allow users to dynamically add and manage database connections (PostgreSQL, MySQL, Firestore, etc.)
- Website Source Management: Provide a settings page to manage website sources, including adding, activating/deactivating, and deleting sources. Auto-generate logos for each site and display relevant information.
- Interactive Chat Interface: Implement a modern chat interface for users to ask questions and receive answers from the AI agent, complete with a customizable agent name, conversation history, and responsive design.
- User Authentication and Roles: Implement a basic authentication system to manage admin/user roles.

## Style Guidelines:

- Primary color: Deep Indigo (#6639A6) to convey intelligence and sophistication.
- Background color: Very light grey (#F2F0F7), almost white, for a clean, modern look.
- Accent color: Electric Purple (#BF57E0), a brighter variation of the primary hue, for highlights and CTAs.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel; body text: 'Inter' (sans-serif) for readability.
- Code font: 'Source Code Pro' (monospace) for displaying database connection strings.
- Use clean, minimalist icons to represent website sources and database types.
- Implement smooth, subtle animations for loading states and transitions.
